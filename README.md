Estructura básica o esquelo de sistema bibliográfico de citado y de composición de bibliogrfía, compuesto íntegramente en LaTeX con la fuente [AltStandard](https://github.com/Ioannes-Brclds/AltStandard.git). Se puede cambiar la estructura como se considere necesario, y hacer un "fork" con modificaciones con la condición de cambiar el nombre. Si tiene alguna sugerencia, compártala.
