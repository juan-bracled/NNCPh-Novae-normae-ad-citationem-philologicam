Estructura básica o esquelo de sistema bibliográfico de citado y composición de bibliografías, compuesto íntegramente para LaTeX, pero que puede usarse, siguiendo la estructura, para otros programas como Microsoft Word. El documento se compuso con la fuente [AltStandard](https://github.com/Ioannes-Brclds/AltStandard.git). El código en los archivos .bbx, .blx, .cbx y .dbx fue producido por la Inteligencia Artifical (Google Gemini) a partir de indicaciones específicas del diseño técnico del sistema de citación. Se puede cambiar la estructura como se considere necesario y hacer un "fork" con modificaciones, con la condición de cambiar el nombre del sistema (se puede cambiar la licencia también, siempre y cuando se respete que sea "código abierto"). Si tiene alguna sugerencia, compártala.
